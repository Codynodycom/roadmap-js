# edu

Образовательное Vue3-приложение.

Реализованный функционал (v.1.0):

* Навигация
* Подгрузка постов с JsonPlaceholder
* Создание(добавление) и удаление постов в интерфейс
* Поиск и 2 типа сортирвки постов
* Протестирован Store (Vuex)
* Анимации
